# SHOSHO MAKEUP - موقع تجميل فاخر

موقع ويب حديث وعصري لعلامة SHOSHO MAKEUP، يقدم تجربة تسوق فاخرة مع تصميم احترافي وواجهة سلسة.

## ✨ المميزات

### التصميم والواجهة
- **تصميم فاخر وعصري**: ألوان احترافية مع تدرجات وردية وذهبية
- **واجهة سلسة**: انتقالات ناعمة وتأثيرات بصرية جميلة
- **استجابة كاملة**: يعمل بشكل مثالي على جميع الأجهزة (موبايل، تابلت، ديسكتوب)

### الأقسام الرئيسية
1. **الهيدر**: شعار فاخر مع قائمة تنقل وسلة تسوق
2. **قسم البطل (Hero)**: عرض جذاب مع دعوة للعمل
3. **المنتجات**: عرض شبكي لـ 10 منتجات مع صور عالية الجودة
4. **قصتنا**: معلومات عن العلامة التجارية مع إحصائيات
5. **التواصل**: معلومات الاتصال وموقع المتجر
6. **التذييل**: روابط سريعة ومعلومات التواصل

### نظام السلة
- إضافة المنتجات إلى السلة بسهولة
- تحديث الكميات وحذف المنتجات
- عرض الإجمالي بشكل فوري
- واجهة منبثقة سلسة

## 🛠️ التقنيات المستخدمة

- **React 19**: مكتبة JavaScript لبناء واجهات المستخدم
- **Vite**: أداة بناء سريعة وحديثة
- **CSS3**: تنسيقات حديثة مع متغيرات CSS
- **JavaScript ES6+**: كود حديث وفعال

## 🚀 البدء السريع

### المتطلبات
- Node.js (الإصدار 16 أو أحدث)
- npm أو yarn

### التثبيت

```bash
# استنساخ المشروع أو الدخول إلى مجلد المشروع
cd shoshomakeup

# تثبيت المكتبات
npm install
```

### التطوير

```bash
# بدء خادم التطوير
npm run dev

# سيكون الموقع متاحاً على http://localhost:5173
```

### البناء للإنتاج

```bash
# بناء النسخة الإنتاجية
npm run build

# سيتم إنشاء مجلد dist/ يحتوي على الملفات الجاهزة للنشر
```

### معاينة النسخة الإنتاجية

```bash
npm run preview
```
