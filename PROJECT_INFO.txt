🚀 خطوات النشر السريعة:
================================================================================

1. تثبيت المكتبات:
   npm install

2. بناء النسخة الإنتاجية:
   npm run build

3. نشر ملفات dist/ على خادمك:
   - استخدم Netlify أو Vercel للنشر السهل
   - أو انسخ ملفات dist/ إلى خادمك الخاص

4. تكوين الخادم:
   - استخدم Nginx أو Apache
   - فعّل HTTPS باستخدام Let's Encrypt
   - اتبع التعليمات في DEPLOYMENT.md
